A personal finance tracking web application built as a demo project.

**Live Demo:** (https://finance-tracker-delta-seven.vercel.app/)

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS v4
- shadcn/ui
- Recharts
- Framer Motion
- Sonner (toasts)

## Features

- 📊 Dashboard with balance, income & expense summary
- 💸 Add, delete & filter transactions
- 📂 Category-based spending breakdown
- 📈 Pie chart & monthly bar chart analytics
- 🎯 Monthly budget goals with progress bars & alerts
- 🧾 Recurring bills tracker with due date countdown
- 🔔 Notification bell for upcoming bills & over-budget alerts
- 🌙 Dark / light mode toggle
- 👤 Profile settings with custom icons
- 💾 Fully persisted via localStorage
